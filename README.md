# UBRAIN Website

**UBRAIN**  
Ungauged Basin Rainfall Analysis and Intelligence Network

A research platform for sub-daily rainfall generation, IDF analysis and hydrological intelligence for ungauged river basins.

## Developed by

Hari Prakash and Dr. Pramod Soni  
Department of Civil Engineering  
Indian Institute of Technology (BHU), Varanasi

## Free GitHub Pages Deployment

1. Create a GitHub account.
2. Create a new repository named:

   `ubrain-india.github.io`

3. Upload these files:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`

4. Go to:

   Settings → Pages

5. Select:
   Branch: `main`
   Folder: `/root`

6. Your free website will open at:

   `https://ubrain-india.github.io`

## Notes

This is Version 1: static research portal only.

The actual rainfall generation engine should not be uploaded publicly until publication and validation are completed.
